# QPlayer

播放同一WIFI网络中电脑共享的视频文件，支持wifi协议的视频文件传输，支持网页观看视频。

## 使用

在app设置中打开Wi-Fi文件传输的开关，在电脑浏览器中访问：如"http://192.168.1.34:63425"，然后选择文件后点击上传，确保上传中不要关闭本应用并在同一wifi环境下使用本软件。
